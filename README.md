# Cinema Booking App

## Description

This is a simple Flutter application that simulates a cinema booking system.

The user can:

* View movies
* Select date and time
* Choose seats
* View ticket information

## Features

* Movie list screen
* Search functionality (filter)
* Date and time selection
* Seat selection system
* Ticket display

## Technologies

* Flutter
* Dart

## How to Run

1. Open the project
2. Run:
   flutter pub get
3. Run the app:
   flutter run
